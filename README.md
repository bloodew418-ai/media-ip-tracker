# media-ip-tracker

漫画・アニメ・映画・ドラマ・小説などを、作品IP単位でまとめて管理する個人用Webアプリです。

## 目的

単体の作品記録ではなく、1つの作品グループに対して、原作漫画、アニメ化、実写映画、実写ドラマ、小説、スピンオフなどを紐づけて管理します。

## 公開形態

GitHub Pages で公開できるよう、アプリ本体は `/docs` 配下に配置しています。`docs/index.html` を開くとローカルでも利用できます。

### GitHub Pages 公開URL

- 公開URL: `https://bloodew418-ai.github.io/media-ip-tracker/`

### GitHub Pages 設定手順

1. GitHub のリポジトリ画面で Settings を開きます。
2. Pages を開きます。
3. Build and deployment で Source に `Deploy from a branch` を選択します。
4. Branch に `main` を選択します。
5. Folder に `/docs` を選択します。
6. Save を押します。

### 公開後の確認項目

- ページが開けること。
- 作品グループ一覧が表示されること。
- サンプルデータ読込ができること。
- `localStorage` 保存ができること。
- JSONエクスポート/インポートができること。
- 検索リンクが別タブで開くこと。
- 390px/360px幅で横スクロールしないこと。

## 主な機能

- 作品グループの新規作成、編集、削除
- 代表タイトル、別名、タグ、作品全体の状態、ネタバレなし/ありメモの管理
- 媒体版の追加、編集、削除
- 媒体タイプ、役割、年、状態、評価、メモの管理
- 見る/買うリンクの手動登録
- 出典情報の手動登録
- 作品タイトルに基づく検索リンク自動生成
- タイトル検索、タグ検索、媒体タイプフィルター、作品状態フィルター
- localStorage保存
- JSONエクスポート/インポート
- サンプルデータJSON

## データ方針

外部サービスの情報は、以下を分けて扱います。

1. 将来的にAPIで確認できる情報
2. 検索リンクとして案内する情報
3. ユーザーが手動登録した情報
4. 未確認の情報

配信中・販売中は原則として断定せず、確認済み、検索リンク、手動登録、未確認、利用不可を分けて表示します。
