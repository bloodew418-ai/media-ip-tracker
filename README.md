# media-ip-tracker

漫画・アニメ・映画・ドラマ・小説などを、作品IP単位でまとめて管理する個人用Webアプリです。

## 目的

単体の作品記録ではなく、1つの作品グループに対して、原作漫画、アニメ化、実写映画、実写ドラマ、小説、スピンオフなどを紐づけて管理します。

## 主な機能

- 作品グループ管理
- 媒体版管理
- 状態管理
- タグ管理
- ネタバレなしメモ / ネタバレありメモ
- 見る/買う検索リンク
- 出典管理
- 手動リンク管理
- JSONエクスポート/インポート
- localStorage保存

## データ方針

外部サービスの情報は、以下の3段階で扱います。

1. APIで確認できる情報
2. 検索リンクとして案内する情報
3. ユーザーが手動登録した情報

配信中・販売中は原則として断定せず、確認済み、検索リンク、手動登録、未確認を分けて表示します。
