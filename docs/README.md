# Media IP Tracker docs app

`/docs` は GitHub Pages で公開する静的Webアプリ本体です。

## 公開URL

- 公開URL: `https://bloodew418-ai.github.io/media-ip-tracker/`

## 方針

- HTML/CSS/JavaScript のみで実装しています。
- 保存先はブラウザの `localStorage` です。
- APIキーが不要な公開API/公開データのみ自動検索に使います。APIキーが必要な外部API連携、スクレイピング、ログイン、クラウド同期は行いません。
- 検索リンクは各サービスの検索結果を別タブで開くためのもので、配信中・販売中とは断定しません。
- 手動登録したリンクや出典は、確認状態と最終確認日を分けて管理します。

## 使い方

1. `index.html` をブラウザで開きます。
2. 「作品グループ追加」から代表タイトル、別名、タグ、状態、メモを登録します。
3. 詳細画面で媒体版を追加し、媒体タイプ、役割、年、状態、評価、メモを記録します。
4. 媒体版編集画面で、見る/買うリンクと出典を `|` 区切りで手動登録します。
5. 「自動検索」から作品名を入力し、Google Books / AniList / Wikidata の候補を検索できます。
6. 検索結果カードから、新規作品グループとして保存、既存作品グループへの媒体版追加、除外ができます。
7. データ管理画面からJSONエクスポート/インポート、サンプルデータ読込ができます。

## 自動検索の注意

- Google Books、AniList、Wikidata の公開エンドポイントをブラウザから呼び出します。
- APIキーが必要な TMDb や楽天ブックスAPIは未対応です。将来 Cloudflare Worker 等のプロキシ経由で対応する想定です。
- 1つの検索元が失敗しても他の検索結果は表示します。
- 自動検索結果は候補であり、配信中/販売中とは断定しません。

## スマホ対応

390px/360px 幅でも主要操作がしやすいよう、フォームとカードは1カラム表示になり、ボタンは押しやすい高さを確保します。
