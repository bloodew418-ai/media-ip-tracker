# Media IP Tracker v2

`/docs` はGitHub Pagesで公開する検索ファースト版のWebアプリです。

## v2の方針

- v2は登録式の管理アプリではなく、作品名検索を最初の主導線にした検索ファーストUIです。
- 作品名を入力して、漫画・アニメ・ドラマ・映画・小説を横断検索します。
- 検索結果ページでは、作品名、目次、媒体別候補、日本語で確認しやすい探すリンク、補助扱いの出典リンクを保存なしで閲覧できます。

- 検索結果ページは候補を並べるだけではなく、作品判定サマリーを先頭に表示します。
- 最有力候補、媒体展開、外部確認リンクを見て、次に見る/読む作品を判断できるようにします。
- 作品判定は完全な自動判定ではなく、公開データからの候補整理です。主導線は日本語向けの「確認する」「探す」検索リンクで、英語のデータ出典リンクは補助扱いです。
- AniList / Google Books / Wikidata はAPIキーなしで検索します。
- TMDbは任意です。映画・ドラマ検索を強化したい場合だけ、Web画面の「設定/API」でRead Access Tokenを保存します。
- 保存は検索結果を見た後の任意操作です。保存すると、お気に入りや進捗状態をlocalStorageに保持できます。
- TMDbトークン、保存済み作品、インポートデータはブラウザのlocalStorageに保存します。
- お気に入りは作品IPグループの `isFavorite`、進捗状態は媒体候補ごとの `status` として分けて保存します。
- スクレイピング、ログイン、課金、クラウド同期、AIレコメンドは実装していません。
- NetflixやAmazonなどは「探す」ための検索リンクであり、配信中・販売中とは断定しません。日本国内で探しやすい導線を優先します。
- YouTubeリンクは検索リンクであり、公式性を保証するものではありません。

## 画面

1. 検索トップ（主導線は作品名検索のみ）
2. 検索結果ページ（保存不要で媒体別候補を閲覧）
3. 作品詳細 / 保存済みページ（検索後の補助管理）
4. 設定 / APIキー管理
5. データ管理

v1の `/docs` は `../docs-v1` に退避しています。
